# GAD酶基因组预测工具

这是一个专业的GAD酶（谷氨酸脱羧酶）全基因组预测工具，可以从核苷酸序列预测潜在的GAD酶基因。

## 功能特点

### 1. 核苷酸序列处理
- 支持基因组核苷酸序列输入（DNA序列）
- 自动识别所有6个阅读框（正向3个 + 反向3个）
- 自动转换核苷酸序列为蛋白质序列

### 2. GAD酶特征识别
- **PLP结合位点检测**：G[ST]G[KR][DE]等保守模式
- **催化活性位点检测**：K[ST][DE]、[KR][ST][DE]K等模式
- **谷氨酸结合位点检测**：[FY][KR][DE]等模式
- **保守区域识别**：KK[DE]、[KR][KR][DE]等GAD特有模式

### 3. 专业的评分算法
- **结构评分**：基于氨基酸组成和长度
- **模式评分**：基于保守模式匹配
- **PLP结合位点加分**：识别PLP结合位点的重要性
- **活性位点加分**：识别催化活性位点的重要性

## 使用方法

### 基本用法
```bash
python gad_genome_predictor.py -g LR1.fna -o predictions
```

### 参数说明
```
-g, --genome     基因组核苷酸序列文件（FASTA格式）
-o, --output     输出目录（默认：gad_genomic_predictions）
-t, --threshold  筛选阈值（默认0.6）
```

### 示例
```bash
# 基本分析
python gad_genome_predictor.py -g LR1.fna -o results

# 使用更高阈值筛选高质量候选
python gad_genome_predictor.py -g LR1.fna -o results -t 0.7

# 使用较低阈值筛选更多候选
python gad_genome_predictor.py -g LR1.fna -o results -t 0.5
```

## 输出文件

### 1. gad_genomic_predictions.tsv
候选GAD酶基因列表（TSV格式），包含以下信息：
- Region_ID：基因组区域ID
- Frame：阅读框编号
- GAD_Score：GAD酶可能性评分
- Length：蛋白质长度
- Structural_Score：结构评分
- Motif_Score：保守模式评分
- PLP_Matches：PLP结合位点匹配数
- Active_Matches：活性位点匹配数
- Motif_Count：总保守模式匹配数
- 氨基酸组成：hydrophobic、hydrophilic、charged等
- Protein_Sequence：蛋白质序列片段

### 2. gad_genomic_report.txt
详细的分析报告，包含：
- 分析概况
- Top候选列表
- 统计信息
- 阅读框分布

## GAD酶的关键特征

### 生物学特征
1. **PLP结合位点**：G[ST]G[KR][DE]（必需的辅因子结合位点）
2. **催化活性位点**：包含K、S、E等氨基酸
3. **保守序列**：KK[DE]、K[KR]X[DE]K等
4. **长度范围**：通常在400-600氨基酸之间
5. **氨基酸组成**：带电氨基酸（K、R、D、E）比例较高

### 评分系统
1. **结构评分（30%）**：基于长度和氨基酸组成
2. **保守模式评分（35%）**：基于PLP结合位点、活性位点等
3. **PLP结合位点加分（25%）**：PLP结合位点的存在是GAD酶的关键特征
4. **活性位点加分（10%）**：催化活性位点的存在增加可信度

## 阈值设置建议

### 高质量候选（严格筛选）
阈值：0.7-0.8
适合：筛选高质量、可信度高的GAD酶候选

### 中等质量候选（中等筛选）
阈值：0.6-0.7
适合：筛选可信度较高的GAD酶候选

### 宽松筛选（全面搜索）
阈值：0.5-0.6
适合：搜索所有可能的GAD酶基因

## 验证建议

### 1. PLP结合位点验证
检查候选序列是否包含典型的PLP结合位点模式

### 2. 活性位点验证
检查候选序列是否包含催化活性位点

### 3. 同源性验证
与已知GAD酶序列进行比对

### 4. 功能验证
通过实验验证GAD酶活性

## 注意事项

### 1. 基因组序列格式
输入必须是核苷酸序列（DNA序列），可以是：
- 基因组序列（多个contigs）
- 基因序列（单个基因）
- 转录本序列

### 2. 阈值设置
阈值越高，筛选越严格，候选数量越少
阈值越低，筛选越宽松，候选数量越多

### 3. 准确性
本工具基于保守模式识别，可能存在假阳性
建议结合其他方法进行验证

## 与其他工具的区别

### 优点
1. **专业针对GAD酶**：专注于谷氨酸脱羧酶的识别
2. **核苷酸序列支持**：直接从基因组序列预测
3. **保守模式识别**：基于GAD酶的真实保守序列
4. **PLP结合位点检测**：关键辅因子结合位点的识别

### 局限性
1. **假阳性可能**：保守模式识别可能存在误报
2. **需要验证**：建议结合实验验证
3. **依赖模式**：对保守模式的准确性依赖较高
