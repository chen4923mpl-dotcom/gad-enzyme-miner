# 贡献指南

欢迎为GAD酶挖掘工具贡献代码！

## 如何贡献

### 1. 报告问题
如果你发现任何问题，请：
1. 在GitHub Issues中创建问题报告
2. 描述具体问题
3. 提供相关代码或数据示例

### 2. 提交改进
如果你想改进代码：
1. Fork本项目
2. 创建分支进行开发
3. 提交Pull Request

### 3. 添加功能
欢迎添加以下功能：
- 机器学习分类器
- 结构预测集成
- 功能域注释
- 可视化工具

## 代码规范

### 风格指南
- 使用Python PEP 8规范
- 函数名使用小写字母和下划线
- 类名使用驼峰命名法
- 注释清晰易懂

### 测试要求
- 每个新功能都应包含测试用例
- 现有功能的修改不应破坏现有测试
- 测试覆盖率应保持在80%以上

## 文档要求

### 添加文档
- 新功能需要相应的文档
- 更新README.md中的使用说明
- 添加示例代码

### 文档格式
- 使用Markdown格式
- 代码示例应清晰易懂
- 添加必要的注释

## 许可证

本项目使用MIT许可证。贡献的代码也将遵循该许可证。
